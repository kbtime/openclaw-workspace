## 定时任务优化

### 问题：LLM 1305 错误
**原因**：OpenClaw cron 任务的 `agentTurn` 模式会启动 Agent 会话，调用 LLM

### 解决方案
将任务迁移到系统 crontab（不调用 LLM）：

**OpenClaw 禁用的任务**：
- 刷新飞书 User Token
- 刷新飞书 Tenant Token
- HDP 12:00/18:00/22:00 同步

**系统 Crontab 保留**：
```bash
0 12 * * * /tmp/hdp-sync-12.sh
0 18 * * * /tmp/hdp-sync-18.sh
0 22 * * * /tmp/hdp-sync-22.sh
0 * * * * ~/.openclaw/workspace/scripts/feishu-token-keepalive.sh
```

**OpenClaw 保留**：
- 同步 Dashboard 执行记录（每 5 分钟）
- OpenClaw 每日自动更新
