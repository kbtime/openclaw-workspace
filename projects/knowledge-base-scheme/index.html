🔧 知识库管理流程（Git）

知识库使用 Git 管理，支持版本控制和团队协作

# 1. 克隆知识库
git clone https://github.com/xxx/knowledge-base.git
cd knowledge-base

# 2. 添加/修改文档
vim policies/请假制度.md

# 3. 提交更改
git add .
git commit -m "更新请假制度"

# 4. 推送到远程
git push origin main
