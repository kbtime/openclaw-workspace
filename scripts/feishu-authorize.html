🔐 飞书授权 - 获取 Refresh Token

步骤 1: 点击授权链接

点击下方按钮，跳转到飞书授权页面：

🔓 开始授权

授权范围：

  • auth:user.id:read - 读取用户 ID
  • offline_access - 获取 refresh_token（必须）

步骤 2: 扫码授权

在飞书授权页面：

  1. 使用飞书扫码
  2. 确认权限列表包含 offline_access
  3. 点击"同意授权"

步骤 3: 获取 Token

授权成功后，会跳转到回调页面。你需要：

  1. 从 URL 中复制 code 参数
  2. 运行下面的脚本换取 access_token 和 refresh_token
⚠️ 注意：授权码（code）有效期只有 5 分钟，且只能使用一次！

步骤 4: 换取 Token

复制授权码后，运行：

cd ~/.openclaw/workspace
node scripts/get-token-from-code.js <你的授权码>
📋 前置检查：
1. 确保飞书开放平台 → 权限管理 → 已开通 offline_access
2. 确保飞书开放平台 → 安全设置 → 已开启"刷新 user_access_token"开关
3. 确保应用已发布（修改配置后需要发布）
✅ 成功后：
你会获得 access_token 和 refresh_token，然后运行：
node scripts/init-feishu-token.js <access_token> <refresh_token>
