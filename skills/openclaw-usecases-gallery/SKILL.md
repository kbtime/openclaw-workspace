# OpenClaw Usecases Gallery

本 Skill 收录了 13 个实用的 OpenClaw AI Agent 使用案例，涵盖内容创作、生产力工具、个人管理等多个领域。

## 使用案例列表

### 1. Daily YouTube Digest
每日 YouTube 视频摘要，从你关注的频道获取最新内容并生成关键洞察。

**需要的 Skill**: youtube-full

### 2. Todoist Task Manager
将 Agent 的任务状态同步到 Todoist，可视化复杂工作流的进度。

**需要的工具**: Todoist API, Bash 脚本

### 3. Multi-Channel Personal Assistant
多频道个人助手，整合 Telegram、Slack、Google Workspace、Todoist、Asana。

**需要的 Skill**: gog CLI, Slack, Todoist, Asana

### 4. Personal CRM with Auto Discovery
自动发现并维护个人 CRM，跟踪联系人和互动历史。

**需要的 Skill**: gog CLI (Gmail + Calendar)

### 5. Knowledge Base (RAG)
个人知识库，保存文章、推文、视频内容并支持语义搜索。

**需要的 Skill**: knowledge-base, web_fetch

### 6. YouTube Content Pipeline
YouTube 内容创作管道，自动发现视频创意并生成研究卡片。

**需要的 Skill**: web_search, x-research-v2, knowledge-base, Asana

### 7. Health & Symptom Tracker
健康症状追踪器，记录饮食和症状并分析潜在触发因素。

**需要的工具**: Cron, Telegram

### 8. X Account Analysis
X/Twitter 账号分析，深入了解帖子表现和内容质量。

**需要的 Skill**: Bird (X/Twitter)

### 9. AI-Powered Earnings Tracker
AI 驱动的财报追踪器，监控科技公司财报并生成摘要。

**需要的工具**: web_search, Cron

### 10. Daily Reddit Digest
每日 Reddit 摘要，从你关注的 subreddit 获取热门帖子。

**需要的 Skill**: reddit-readonly

### 11. Phone-Based Personal Assistant
基于电话的个人助手，通过语音与 AI 交互。

**需要的 Skill**: ClawdTalk

### 12. Inbox De-clutter
收件箱整理，自动汇总新闻邮件并生成摘要。

**需要的 Skill**: Gmail OAuth

### 13. Overnight Mini-App Builder
夜间迷你应用构建器，让 AI 在你睡觉时构建小型应用。

**特点**: 无需特定 skill，直接提示 AI 构建 MVP
