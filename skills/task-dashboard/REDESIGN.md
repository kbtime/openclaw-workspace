### 视觉风格
- 字体: Inter, -apple-system, sans-serif
- 圆角: 8px
- 间距: 16px / 24px
- 阴影: 极简 (0 1px 2px rgba(0,0,0,0.3))

## 需要修改的文件

1. `public/index.html` - 新布局结构
2. `public/style.css` - 新样式
3. `public/app.js` - 更新渲染逻辑

## API 端点 (保持不变)
- GET /api/status - 系统状态
- GET /api/cron - 定时任务列表
- GET /api/records - 执行记录 (分页)
- GET /api/tasks - 任务列表

## 功能要求
1. 顶部系统状态横条，显示 Gateway、User Token、Tenant Token、Nginx、Docker 状态
2. 左侧定时任务卡片，显示任务名称、cron 表达式、下次执行时间、状态
3. 右侧执行记录卡片，支持分页 (每页 10 条)
4. 底部显示最后更新时间
5. WebSocket 失败时自动切换到轮询模式
6. 响应式设计，移动端适配
